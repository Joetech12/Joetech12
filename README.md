## Hi there 👋  
I specialise in building high-performance, scalable, and user-focused web and mobile applications. My work is centered on delivering clean, maintainable solutions that combine strong engineering principles with modern frontend technologies.
 
I have experience developing production-ready applications across web and mobile, with a focus on performance, responsiveness, and seamless user experience. I also leverage Firebase to design and manage secure, real-time, and scalable backend systems.
 
🧠 Core Technologies:

 Next.js, React, React Native, TypeScript, JavaScript, React Query, Tailwind CSS, Zustand, Jotai, Shadcn UI, Firebase, WordPress, WooCommerce
 
💼 What I Do:

 1.  Design and build scalable web applications (frontend and backend). 
 2.  Develop modern, responsive user interfaces.
 3.  Architect and manage real-time data systems with Firebase. 
 4.  Implement secure authentication and access control. 
 5.  Optimize performance, SEO, and overall user experience. 
 
I have worked across multiple industries including real estate, SaaS, eCommerce, healthcare, and education, delivering solutions from MVPs to fully scaled platforms.
 
I take ownership of projects end-to-end, with a strong focus on code quality, system design, and long-term maintainability.
